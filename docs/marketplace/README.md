# ATiQ AI for VS Code

[![Marketplace Version](https://img.shields.io/visual-studio-marketplace/v/atiq.atiq-ai?color=brightgreen)](https://marketplace.visualstudio.com/items?itemName=atiq.atiq-ai)

**ATiQ AI** is your professional-grade AI coding partner, designed for speed, security, and deep project understanding.

## Key Features

- **🚀 Real-time Chat**: Ask questions about your entire project, not just the active file.
- **✨ Magic Inline Edits (Cmd+K)**: Highlight code and describe changes. ATiQ generates a smart diff for you to review.
- **🔍 Deep Symbol Indexing**: ATiQ builds a high-performance index of your symbols for accurate codebase-wide context.
- **🛡️ Secure by Design**: Built-in telemetry controls and secret detection to keep your code private and safe.
- **🛠️ One-click Fixes**: Instantly fix bugs, optimize performance, or generate tests.

## Installation

1. Install the extension from the [VS Code Marketplace](https://marketplace.visualstudio.com/items?itemName=atiq.atiq-ai).
2. Use `Cmd+Shift+A` (Mac) or `Ctrl+Alt+A` (Windows) to open the ATiQ sidebar.
3. Sign in using your Magic Link to unlock full features.

## Usage Tips

- **Context is King**: Use `@` in chat to mention specific files or symbols.
- **Iterative Edits**: Use Cmd+K multiple times to refine code until it's perfect.
- **Review Mode**: ATiQ shows all changes in a dedicated diff view so you're always in control.
