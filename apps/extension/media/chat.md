# ATiQ AI Chat

You can chat with ATiQ about your current file or the entire workspace.

### Key features:

- **Project Awareness**: Ask "Where is the auth logic?" or "How do I add a new route?".
- **Code Generation**: "Write a React component for a data table".
- **Debugging**: "Why is this function returning undefined?".

### Shortcuts:

- `Cmd+Ctrl+A`: Open Chat panel
- `Cmd+Shift+L`: Send selection to chat
